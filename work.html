Personal Portfolio:

This is my personal portfolio, which was designed and developed by myself. I used various different tools to create the final product.

The technologies/languages used are as followed:
- HTML.
- CSS.
- SCSS.
- JavaScript.
- Moqups (Online Mockup, Wireframe and UI Prototyping Tool).
- VSCode.
- Photoshop.

This website was developed after the completion of 'Responsive Web Design', 'JavaScript Algorithms' and 'Front-End Libraries' units in FreeCodeCamp. I decided to put my knowledge of HTML, CSS and JavaScript to the test and my end product is the website you're currently browsing. My portfolio website has recently been updated, incorporating SASS workflow, allowing to keep things more organised and creating style sheets much more efficiently.
